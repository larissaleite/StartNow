Clone Repository GitHub

Create repository in https://github.com/<username>

In the command line, go to the folder where you want to create your repository folder

> git clone https://github.com/<username>/<repository name>.git

Add and edit a README file:

> touch README.md

> vi README.md

Make your first commit:

> git add README.md

> git commit -m "First commit"

> git push origin master

New Repository from the Command LineGitHub

Create repository in https://github.com/<username>

Create folder in your local machine, with the same name as the repository

Reach the folder using the command line and the type the following:

> git --global user.name <username>

> git --global user.email <email>

This command is useful so that you don't have to type in your username and password everytime you push new changes

> git init

> git remote add origin https://github.com/<username>/<repository name>

Creating Branches GitHub

Branches can be created either from the command line or on GitHub

1. Creating branches on GitHub

Create new branch in https://github.com/<username>/<repository name>

For you local repository to become aware of the new branch created, using the command line, type:

> git fetch

To switch to the created branch:

> git checkout <branch name>

To merge with master (get changes from master):

> git merge master
